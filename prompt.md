# Prompt for LLM Coding Agent: Maintain rowanemilia.com Hugo Site

This site has been fully migrated from WordPress (https://www.rowanemilia.com/) to a self-hosted Hugo static site. Read `spec.md` for the complete specification. Below is the current build and maintenance workflow.

## Prerequisites

- `hugo` (extended version)
- `python3`
- Docker (optional, for containerized builds)

## Build & Serve

```bash
# 1. Generate per-month archive pages from post dates
python3 gen_archives.py

# 2. Build the static site
hugo --disableKinds=taxonomy

# 3. Serve locally
hugo server --disableKinds=taxonomy
# or serve the public/ directory:
python3 -m http.server 1314 --directory public
```

## Docker Build

```bash
docker build . --no-cache --file Dockerfile --tag rowanemilia:latest
docker run -p 8080:80 rowanemilia:latest
```

The Dockerfile runs `gen_archives.py` before `hugo` automatically.

## Adding New Content

1. Create a new post as Markdown in `content/posts/YYYY/MM/` with proper front matter
2. Place images in `static/images/YYYY/MM/`
3. Run `python3 gen_archives.py` to update archive pages for any new month
4. Rebuild with `hugo --disableKinds=taxonomy`

## CI/CD

A GitHub Action at `.github/workflows/docker-image.yml` builds and pushes a Docker image to GitHub Container Registry on pushes to `main`.

## Key Files

| File | Purpose |
|------|---------|
| `hugo.yaml` | Site configuration (YAML) |
| `convert.py` | Initial WordPress→Hugo converter (one-time use) |
| `html2md.py` | Converts HTML post bodies to Markdown |
| `gen_archives.py` | Generates per-month archive `_index.md` files |
| `data/comments.json` | 154 comments from WordPress |
| `themes/rowan-wp/` | Custom Hugo theme (diary-lite port) |

## Theme Structure

```
themes/rowan-wp/
├── theme.yaml
├── static/
│   ├── css/style.css        # diary-lite CSS reproduction
│   ├── js/lightbox.js       # Vanilla JS lightbox
│   └── img/                 # 9 theme background images
└── layouts/
    ├── baseof.html
    ├── _default/
    │   ├── index.html       # Paginated post list
    │   ├── single.html      # Single post/page with comments
    │   └── list.html        # Category/archive listing
    ├── archives/list.html   # Per-month archive pages
    └── partials/
        ├── header.html
        ├── sidebar.html     # Menu, Categories, Archives
        └── footer.html      # Copyright only
```

## Architecture Notes

- Sidebar uses **flexbox** (`display: flex; flex-direction: row-reverse`) for reliable left-sidebar layout
- Post content is **Markdown** (converted from WordPress HTML via `html2md.py`)
- YouTube embeds preserved as raw HTML `<iframe>`
- Lightbox images use `class="lightbox"` / `class="lightbox-gallery"`
- Per-month archive pages at `/archives/YYYY/MM/` are generated by `gen_archives.py`
- No external WordPress URLs remain in rendered output
- No deprecation warnings on build
